Function photon-current

Package:

PHOTON-FLUX.PHOTON-CALCULATIONS

Syntax: 

  photon-current POWER WAVELENGTH → result

Arguments and Values:

  • POWER : a T
  • WAVELENGTH : a T
  • result : a T.

Description:

Photon current Arguments and Values: POWER : beam power in Watt WAVELENGTH : light wavelength in Meter PHOTON-CURRENT : Ampere Notes: For verification, use "Calculator for Photodiodes" on https://www.rp-photonics.com/photodiodes.html
