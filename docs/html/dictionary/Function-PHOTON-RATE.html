Function photon-rate

Package:

PHOTON-FLUX.PHOTON-CALCULATIONS

Syntax: 

  photon-rate POWER WAVELENGTH → result

Arguments and Values:

  • POWER : a T
  • WAVELENGTH : a T
  • result : a T.

Description:

Number of photons/second Arguments and values: POWER: beam power in Watt WAVELENGTH: light wavelength in Meter
