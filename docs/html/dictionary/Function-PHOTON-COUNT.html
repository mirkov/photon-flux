Function photon-count

Package:

PHOTON-FLUX.PHOTON-CALCULATIONS

Syntax: 

  photon-count BEAM-ENERGY WAVELENGTH → result

Arguments and Values:

  • BEAM-ENERGY : a T
  • WAVELENGTH : a T
  • result : a T.

Description:

Number of photons in a beam Arguments and Values: beam-energy : Beam energy in Joule, a real number wavelength : Photon wavelength in Meter, a real number
